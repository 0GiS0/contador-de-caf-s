# Contador de Cafés

Una app sencilla para contar cuántos cafés tomas al día.

## Características

- ☕ **Contador grande**: Número visible en el centro de la pantalla
- ➕ **Botón +**: Suma 1 al contador de cafés
- 🔄 **Botón Reset**: Reinicia el contador a cero
- 🎨 **Diseño marrón**: Colores cálidos inspirados en el café

## Tecnologías

- Frontend: Vue 3 + TypeScript + Tailwind CSS
- Backend: Spring Boot 3.3 + Java 21
